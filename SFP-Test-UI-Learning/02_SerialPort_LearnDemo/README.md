# 02_SerialPort_LearnDemo
C# WinForm 原生串口通信学习Demo，VS2019 .NET4.8
## 功能清单
1. 自动扫描本机所有COM串口，下拉选择
2. 内置常用波特率 9600/19200/38400/115200
3. 单按钮切换打开/关闭串口，自动释放端口资源
4. 指令下发发送，自动追加换行符适配串口设备协议
5. 异步接收下位机报文，跨线程安全刷新UI
6. 解析SFP模拟报文：提取电压、温度并展示
7. 带时间戳全局日志，自动滚动显示最新消息
8. 全流程异常捕获，弹窗提示串口打开故障

## 依赖调试工具
Virtual Serial Port Driver Pro 9.0
作用：生成成对互通虚拟串口COM3<=>COM4，无硬件闭环测试

## 通信参数默认标准
数据位：8 | 停止位：1 | 校验位：无
