### Task03：03_TestFunction_Demo 批量自动测试+业务功能完整版（本次新增）
#### 核心学习知识点
1. 循环逻辑：`for`循环批量遍历多个SFP模块，模拟自动化轮询测试
2. 状态标记变量 `bool`：`isTesting` 防止重复点击启动测试，`stopTestFlag` 外部终止循环
3. `Thread.Sleep` 指令延时，模拟真实硬件响应间隔；`Application.DoEvents()` 缓解循环界面卡死问题
4. 阈值区间判断：电压、温度上下限合格校验逻辑
5. 界面状态交互：测试合格文字绿色显示、超标不合格红色变色，异常弹窗告警
6. 文件系统全套操作：
   - `Directory` 文件夹创建、`Path` 路径拼接处理
   - `StreamWriter` TXT日志文件本地保存
   - CSV表格文件导出（原生代码无第三方库，Excel直接打开查看测试报表）
7. 配套UI控件实战：
   `ProgressBar`进度条、`StatusStrip`底部状态栏、`Timer`定时器、`ToolTip`鼠标悬浮提示

#### 完整功能清单
1. 底部状态栏实时展示：设备状态、测试运行状态、系统实时时间
2. 一键启动批量自动测试，进度条动态同步展示测试完成进度
3. 一键终止正在运行的测试流程
4. 实时展示当前被测模块参数，自动判定合格/不合格，文字红绿区分
5. 参数超出设定阈值自动弹出异常提示弹窗
6. 防重复点击保护，避免多次启动测试造成数据错乱
7. 程序启动自动按日期分层创建日志文件夹，规范化存储文件
8. 一键保存全部运行日志为TXT文档
9. 一键导出全部测试数据为CSV测试报告，Excel原生打开编辑

#### 运行使用步骤
1. 使用VS2019打开解决方案 `03_TestFunction_Demo.sln`
2. 直接按下 `F5` 启动运行程序
3. 点击【开始批量测试】，自动循环测试10路SFP模块，观察进度条、参数颜色、日志变化
4. 测试中途可点击【停止测试】中断本轮测试
5. 测试结束后，可按需点击「保存TXT日志」「导出CSV测试报告」存储本次数据
6. 程序运行根目录自动生成 `Logs` 文件夹，内部按日期分类存放所有导出文件

#### 预设判定阈值
- 电压合格范围：`3.0V ~ 3.6V`
- 温度合格范围：`0℃ ~ 70℃`
